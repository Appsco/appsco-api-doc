# appsco-api-doc
Appsco API Documentation

Installation
---------------

Follow this steps to start using api doc

- Install vendors 


```bash
$ composer install
```

- Install bower deps

```bash
$ bower install
```

- Create symlink for vendors 

```bash
cd web && ln -s /path/to/bower_components vendor
```

Then start php built in server on port:

```bash
php -S localhost:5000 
```
